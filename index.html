Lista de Contatos - Um Projeto Simples em HTML

Este é um projeto simples de uma lista de contatos feita com HTML e CSS. Ele demonstra como exibir informações de contatos, incluindo nome, foto e e-mail, de maneira organizada e acessível. Objetivo: O projeto tem como objetivo ilustrar a criação de uma interface básica e funcional para uma lista telefônica, utilizando HTML para a estruturação e CSS para o design visual. Ele serve como um exemplo prático de como aplicar estilos e layouts responsivos em pequenos projetos web. Conteúdo: A lista exibe os contatos em um formato simples, com uma imagem de perfil, nome e e-mail de cada pessoa. Há um painel deslizante que permite visualizar múltiplos contatos, tornando o layout prático para diferentes tamanhos de lista.
